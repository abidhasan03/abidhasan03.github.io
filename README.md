# Abid Hasan — Hydejack portfolio

This is a GitHub Pages-ready personal portfolio built on the free Hydejack theme.

## Deploy on GitHub Pages

1. Create a repository named `YOUR-USERNAME.github.io` for a root site, or any repo name for a project site.
2. Upload all files from this folder.
3. In the repository settings, enable **Pages**.
4. If you are using a project repository instead of `YOUR-USERNAME.github.io`, set the `baseurl` in `_config.yml` to `/<repo-name>`.
5. Push to `main`.

## Run locally

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.

## Customize

- Update `_config.yml` for site title, tagline, and navigation.
- Replace `assets/img/logo.svg` and `assets/img/sidebar-bg.svg` with your own graphics if desired.
- Update `resume.md`, `about.md`, `publications.md`, and `projects.md` as your profile evolves.
- Replace `assets/docs/Abid_Hasan_Resume.pdf` whenever you have a newer resume.
